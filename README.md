## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/Idighekere/HNG-Mobile-developments.git
   ```
2. Install dependencies:
    ```bash
    cd simple-app && npm install

3. Run the app:
    ```bash
    npx expo start
